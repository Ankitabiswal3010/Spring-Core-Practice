# SpringCore-practice

A comprehensive Spring Framework Core practice repository showcasing daily coding exercises, mini-projects, and important concepts from beginner to advanced level.

## About the Repository

This repository is created to strengthen my understanding of **Spring Core** and its fundamental concepts. It contains hands-on examples and practice programs covering:

* Spring Bean configuration
* Dependency Injection (Setter & Constructor Injection)
* Autowiring (byName, byType, Constructor)
* Bean scopes and lifecycle methods
* XML-based configuration
* Annotation-based configuration
* Java-based configuration
* Spring Core mini-projects

## Technologies Used

* Java
* Spring Framework (Core)
* Maven
* Lombok
* Eclipse IDE

## How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/Ankitabiswal3010/SpringCore-practice.git
   ```

2. Open the project in **Eclipse IDE**.

3. Update Maven dependencies:

   ```bash
   mvn clean install
   ```

4. Run the main class from the `com.nit.main` package.

The main goal of this repository is to build a strong foundation in **Spring Core** by practicing each concept through real code examples and small projects.
